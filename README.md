# 프로젝트 개요
이 프로젝트는 동시성 테스트를 목적으로 개발된 쇼핑몰 시스템입니다. 다양한 사용자 요청을 동시에 처리하는 성능과 안정성을 검증하기 위해, 상품 목록 조회, 상품 조회, 상품 추가, 상품 수정, 상품 구매 기능을 포함하고 있습니다.
