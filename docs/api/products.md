| HTTP 요청 메서드 | URI                     | 기능               |
|------------------|------------------------|--------------------|
| `GET`            | `/products`            | 상품 목록 조회     |
| `GET`            | `/products/{productId}` | 상품 상세 조회     |
| `POST`           | `/products`            | 상품 추가          |
| `PUT`            | `/products/{productId}` | 상품 수정          |
| `DELETE`         | `/products/{productId}` | 상품 삭제          |
